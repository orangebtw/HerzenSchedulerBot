# Система отслеживания дедлайнов с автоматической привязкой к расписанию на основе Telegram-бота

## Введение
Herzen Organizer - это Telegram-бот, созданный для решения типичной студенческой проблемы, связанной с дедлайнами заданий. Многие студенты бывают, что забывают до последнего момента про какие-либо задания из-за того, что не записали его в заметки или не поставили четкий дедлайн. Мы решили эту боль просто: сделали "умный блокнот", который сам знает ваше расписание университета и автоматически привязывает задания к нужным парам.

Бот работает максимально интуитивно и просто: пишете задание - он предлагает подтвердить предмет (определяет по текущей дате и расписанию), выбираете дедлайн (следующая пара, через одну или конкретная дата) - и всё сохранено. За день и за три часа до дедлайна приходят напоминания, а список задач сортируется по срочности. Это не просто напоминания, а личный секретарь в Telegram, который экономит время и нервы. Проект разработан за полторы недели командой студентов и уже готов к использованию - с возможностью дальнейшего развития в зависимости от обратной связи.

## Описание
Бот построен на проверенном парсинге расписания из старой версии сайта университета (тот же источник, что и в популярном боте с 11 тысячами пользователей). Бот использует асинхронную архитектуру для быстрой обработки сообщений, хранит данные локально и запускает фоновые задачи для своевременных напоминаний. Пользовательский интерфейс реализован через inline-кнопки и машины состояний: добавление, редактирование, удаление и отметка выполнения задач — всё в пару кликов.
### Стек разработки
#### Aiogram
Aiogram - это один из самых популярных фреймворков для разработки ботов в Telegram, написанный полностью на Python. Построен на библиотеке asyncio и использует aiohttp для выполнения HTTP-запросов, что делает его асинхронным и высокоэффективным. Это позволяет боту работать быстро, без зависаний обрабатывать тысячи сообщений одновременно. Одной из причин выбора именно aiogram было то, что в нем можно легко разрабатывать сложные многоуровневые диалоги.

Особенности:
- Асинхронность. Использует async/await и asyncio, что позволяет боту не «зависать» в ожидании ответа и обрабатывать тысячи сообщений параллельно, обеспечивая высокую производительность.
- Масштабируемость. Благодаря асинхронной природе, идеально подходит для ботов с большой аудиторией и высокой нагрузкой.
- Удобные инструменты. Предоставляет хэндлеры (обработчики сообщений) и диспетчер, которые упрощают маршрутизацию и обработку событий от Telegram.
- Современный стек. Работает поверх aiohttp (асинхронный HTTP-клиент) и asyncio, не тащит лишних синхронных зависимостей.
#### sqlite3
sqlite3 - это встроенный модуль Python, который даёт доступ к базе данных SQLite прямо из кода, без отдельного сервера. 

Особенности:
- Встраиваемость - одно из главных преимуществ. В отличие от традиционных СУБД, эта база данных встраивается непосредственно в приложение. Это устраняет необходимость в отдельном сервере, улучшая производительность и упрощая развертывание.

- Самодостаточность. У данной системы управление данными минимализировано и не требует дополнительных компонентов. Это позволяет использовать ее без установки отдельных серверов и приложений, что значительно облегчает процесс разработки и тестирования.

- Поддержка ACID-транзакций. Принцип ACID (атомарность, согласованность, изоляция, долговечность) гарантирует надежность операций с данными. Это особенно важно при разработке приложений на Python, где требуется строгий контроль над целостностью и безопасностью информации.

- Кросс-платформенность. База данных может использоваться на различных операционных системах, что делает ее универсальной. Ее можно эффективно интегрировать в приложения, предназначенные как для Windows, так и для Linux и macOS.

- Компактность и быстрая настройка. Объем кода минимален, что делает её привлекательной для встроенных систем и мобильных устройств. Установка и начальная настройка занимают минимум времени, позволяя разработчикам быстро приступить к работе.

- Поддержка SQL. Несмотря на свою легковесность, система поддерживает большинство стандартных SQL-запросов. Это облегчает работу разработчикам, привыкшим к использованию стандартного языка запросов. 
#### Beautiful Soup
Beautiful Soup - это мощная библиотека для парсинга HTML и XML, которая помогает легко находить элементы на странице, извлекать текст, таблицы и всё нужное, даже если код страницы неидеальный.

Основные возможности:
- Парсинг. Разбирает HTML и XML, в том числе некорректно оформленные документы, создавая структуру данных. 
- Извлечение данных. Позволяет искать и извлекать нужную информацию (текст, ссылки, атрибуты) с помощью фильтров и CSS-селекторов. 
- Навигация. Предоставляет интуитивно понятные методы для перемещения по дереву документа (например, find(), find_all()). 
- Работа с кодировками. Автоматически преобразует данные в Unicode, упрощая работу с разными кодировками. 
- Гибкость. Может работать с различными парсерами, такими как lxml или html5lib, для повышения скорости или обработки сложных случаев. 
#### Requests
Requests - это удобная библиотека для работы с HTTP-запросами в Python. Она позволяет легко отправлять GET, POST и другие запросы, обрабатывать сессии, куки и ошибки.

Основные возможности:
- Простота. Позволяет отправлять запросы в одну строку кода.
- HTTP-методы. Поддерживает все основные методы: GET, POST, PUT, DELETE и другие.
- Работа с данными. Автоматически декодирует JSON-ответы, упрощая обработку данных.
- Сессии. Позволяет сохранять параметры между запросами (например, куки) с помощью объектов Session.
- Аутентификация. Удобная поддержка различных методов аутентификации (Basic, Digest).
- SSL/TLS. Встроенная поддержка защищенных соединений (HTTPS).
- Таймауты. Гибкая настройка времени ожидания ответа.
